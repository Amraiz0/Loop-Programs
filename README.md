# 🖥️ C++ Loop Programs — Interactive Menu

A beginner-friendly C++ project demonstrating core loop concepts through an interactive terminal menu. All 5 programs run from a single file  select any program by entering its number.

---

## 📋 Programs Included

| # | Program | Concept Used |
|---|---------|-------------|
| 1 | Print Numbers 1 to 10 | `for` loop, `range` |
| 2 | Print Even Numbers (2–10) | `for` loop, increment by 2 |
| 3 | Print a Message Multiple Times | `for` loop, string output |
| 4 | Print Star Pattern | Nested `for` loops |
| 5 | Print Numbers in Reverse | `for` loop, decrement |

---

## 🚀 Getting Started

### Prerequisites

- **g++ compiler** (MinGW on Windows, g++ on Linux/Mac)
- Any code editor (VS Code recommended)

### Check if g++ is installed

```bash
g++ --version
```

### Clone the repository

```bash
gh repo clone Amraiz0/Loop-Programs
cd cpp-loop-programs
```

### Compile

```bash
g++ main.cpp -o main.exe
```

### Run

```bash
.\main.exe        # Windows (PowerShell)
./main            # Linux / Mac
```

---

## 🎮 How to Use

```
================================
   C++ Programs Menu
================================
1. Print Numbers 1 to 10
2. Print Even Numbers
3. Print Message Multiple Times
4. Print Stars
5. Print Reverse Order
0. Exit
================================

Choice enter karo: 4
```

Enter a number (0–5) to run any program. Enter `0` to exit.
